###模块目录说明：
*HS*:放置互生模块的类文件 命名前缀：GYHS
*HD*:放置互动模块的类文件  命名前缀：GYHD
*HE*:放置电商模块的类文件  命名前缀：GYHE
*Setting*:放置设置模块的类文件  命名前缀：GYSet
####Main目录下的命名前缀直接使用GY
>> 列子GYHERetailOrderListVC 电商零售订单控制器

...

###图片说明：图片存放在Resource目录下的Assets.xcassets中 所有图片需要提供2x 3x两套图片 使用驼峰标示，使用英语翻译
*Main*:放置主界面使用的图片 命名前缀：main_
*HS*:放置互生模块的图片 命名前缀： gyhs_
*HD*:放置互动模块的图片  命名前缀：gyhd_
*HE*:放置电商模块的图片  命名前缀：gyhe_
*Common*:放置公共类的图片，比如积分图标，箭头等常见的图片 前缀com_
*Setting*：放置设置模块的图片 前缀：gyset_
>> 例子：gyhs_eggImage
